Configuration

  • Settings -> Users&Companies -> Branches.
  • Click on Create button.
  • Fill Branch Information.
  • Choose branch document template:

    Note: The first 4 templates are for the company, so you have to choose between the last 4 templates.

  • Check the created branches under company:

How to Use

  • From Sale/Purchase quotation, click on create

  • Select the branch which the transaction is belong to

  • Print Quotation/ order

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
