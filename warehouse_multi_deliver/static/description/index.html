Warehouse Multi Receipts

  • Features
    • Allow User To Define Default Value For Multi Deliver (For All Purchase Order)
    • Allow User To Add Receipt Lines From Purchase Order Line
    • Allow User To Specify Quantity For Any Picking Operation Type , That Will Receipt Product In It
    • Allow User To Add Multi Receipt Lines For One Purchase Line , And Each Line Can Receipt In Different Picking Operation Type
    • Allow User To Add More Quantity After Receipt Is Done , And Module Will Automated Generate New Picking By Difference Quantity.

Define Multi Receipts As Default For purchase

Ability To Change Multi Receipt Option In Purchase Order , if This Field Un-check System Will Generate One Picking as Odoo Default

Add Purchase Order Lines

After Save Purchase Order Line , Then Add Multu Receipt Lines , By Choose Purchase Order Line

Add Multi Receipt Lines For Purchase Order Lines With Different Picking Operation Type , and specify Quantity For Each One

On Order Confirm , It Will Generate Multi Picking For Receipt Lines(Will Group Them By Picking Operation Type)

Receipt Details

When Picking Validated , It Will Update Recieve Qty In Lines

After Picking Validated , If User Want To Receipt More Quantity, It Can Change PO Line By Add More Quantity, And Then Add Quantity To Receipt Lines Or Add new Receipt Line, Then Module Will Automated Add New Picking For New Quantity
